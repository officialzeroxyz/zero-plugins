# Zero for Qwen Code

How to install Zero in Qwen Code and keep it up to date.

## Install

### From the terminal

```bash
qwen extensions install https://github.com/officialzeroxyz/zero-plugins
```

Restart Qwen Code — Zero sets itself up automatically. Then ask Qwen:
*"Help me set up and test Zero."* It walks you through signing in.

## Staying up to date

- The Zero runner updates at the start of each session.
- To update the extension itself, run:

  ```bash
  qwen extensions update zero
  ```
