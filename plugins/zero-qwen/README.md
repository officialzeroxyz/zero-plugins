# Zero for Qwen Code

Thin Qwen Code extension overlay for the shared Zero plugin.

The installable extension is assembled by `scripts/build-qwen.sh` into
`dist/zero-qwen/`. Source of truth:

- `qwen-extension.json` and `commands/` live here.
- `skills/zero/SKILL.md` and reusable hook scripts are copied from
  `plugins/zero/`.
